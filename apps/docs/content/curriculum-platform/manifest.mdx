# Curriculum platform manifest

- **Role:** Authoring and publishing structured programs, levels, and units for the EFL curriculum; exposes authenticated manager APIs and a **public read API** for published snapshots consumed by the student app.
- **Tech stack:** Next.js Pages Router, credentials session resolved to `CurriculumManager`, `@repo/database`.
- **Prisma models (`curriculum` schema):** `CurriculumManager`, `CurriculumInvite`, `CurriculumProgram`, `CurriculumLevel`, `CurriculumUnit`, `CurriculumPublishSnapshot`, `CurriculumPublishSnapshotUnit`.
