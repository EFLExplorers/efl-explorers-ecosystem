# EFL Explorers Documentation

Welcome to the central documentation hub for the EFL Explorers ecosystem.

This docs app is built with Nextra 4, App Router, and shared workspace packages so guides and UI component docs stay close to production code.

## Start Here

- [Getting Started](/getting-started) for setup and local workflow
- [Teacher Guide](/teacher-guide) for classroom operations
- [API Reference](/api-reference) for integration conventions

## What This Site Covers

- Monorepo workflows with pnpm + Turborepo
- Shared UI components from `@repo/ui`
- Platform operating guidance for teachers
- API request, response, and error formatting standards

## Recommended Path

1. Read **Getting Started** and run the local stack
2. Review **UI Components** before writing new docs pages
3. Use **API Reference** as the source of truth for contract style
