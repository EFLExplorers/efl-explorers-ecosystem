# Code

## Purpose

`Code` renders inline code tokens with consistent styling.

## Import

```tsx
import { Code } from "@repo/ui/code"
```

## Usage Guidance

Use `Code` for short inline commands, identifiers, and file paths. Prefer fenced code blocks for multi-line examples.
