# Getting Started

This section covers setup, local development, docs authoring, and shared UI usage.

## Prerequisites

- Node.js 18+
- pnpm 9+
- Access to the monorepo root

## Quick Start

```bash
pnpm install
pnpm --filter @repo/docs dev
```

Docs run at `http://localhost:3005` (or next open port).
