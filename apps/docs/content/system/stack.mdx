# Ecosystem stack

The **single source of truth** for what belongs in this monorepo’s technology stack is **not** duplicated on this page.

Open the repository file **`.cursor/rules/ecosystem.mdc`** and read **Canonical ecosystem stack**. That section lists approved layers (Next.js, React, Nextra for docs, Prisma, PostgreSQL, pnpm, Turborepo) and **explicit non-goals** (for example, **Astro is not part of this stack**).

When the stack changes, update **`ecosystem.mdc` first**; adjust this page only if the pointer or title needs to change.
