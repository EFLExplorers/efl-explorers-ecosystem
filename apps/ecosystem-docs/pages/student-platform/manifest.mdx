# Student Platform Manifest

- **Role:** Gamified learner experience (missions, planets, Riley / Cass / Emma / Luke as product concepts).
- **Tech Stack:** Next.js App Router, Prisma Client
- **Database Scope:** Primarily `Student`, `StudentUserMapping`, `User`, `Lesson`, `Task`, `Material`, `Bookmark`, `LessonMaterial`, `Event`, `Message`, and `Announcement` (see `schema.prisma`; UI-level “planet/mission” names map to these models in code—confirm in app source before assuming fields).
