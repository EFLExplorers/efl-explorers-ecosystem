# Landing Page API

All API routes live in `apps/landing-page/src/pages/api`.

## Conventions

- Content endpoints require `x-efl-api-key` matching `EFL_API_KEY`.
- Requests should be JSON with `Content-Type: application/json`.

## Docs Index

- `content-endpoints.md` — Content, pages, revalidation, role lookup.
- `auth-endpoints.md` — Registration, login, password reset, SSO.
- `examples.md` — Curl examples for each endpoint.
- `errors.md` — Status codes by endpoint.
