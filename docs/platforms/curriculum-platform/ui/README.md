# Curriculum Platform UI

UI implementation is in `apps/curriculum-platform/src/pages` and uses CSS Modules only.

## Current Screens

- `/login` - Manager sign-in page.
- `/register` - Invite/bootstrap account registration.
- `/dashboard` - Entry point for authoring, publishing, and invite operations.
- `/dashboard/programs` - Program/level/unit workspace.
- `/dashboard/publish` - Publish workflow for level snapshots.
- `/dashboard/invites` - Invite management.
