# Students Platform UI Overview

This app now includes a Milestone 2 UI shell using Next.js pages router and CSS
Modules.

## Key Locations

- Pages: `apps/student-platform/src/pages`
- Shell layout: `apps/student-platform/src/components/student/shell`
- Shared UI state block: `apps/student-platform/src/components/student/StudentStateView.tsx`
- Global styles: `apps/student-platform/src/styles/globals.css`

## Current Scope

- Route-level placeholders for dashboard, lessons, progress, assignments,
  profile, and SSO.
- Shared top-level student navigation.
- Contract-aligned placeholder copy for upcoming API integration.
